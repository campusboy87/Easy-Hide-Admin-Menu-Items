== Description ==

<u>Easy Hide Admin Menu Items</u> is a plugin designed for the convenience of using the WordPress admin panel by hiding unnecessary menu items. On some websites, the admin panel can become cluttered with various menus and settings, and Easy Hide Admin Menu Items is intended to help users easily customize their administrative space.

### Why is it convenient?

On some websites with numerous plugins and extensions, the admin panel can become confusing due to an excess of menus. <u>Easy Hide Admin Menu Items</u> allows you to easily hide the items you don't need, providing cleanliness and convenience in usage.

== Plugin Settings ==

In our plugin, you can locate the settings within the following menu:

<b>Appearance → Admin Menu Settings</b>

Within this menu, you'll find the settings for our plugin.

== Additional Description ==

<u>Easy Hide Admin Menu Items</u> offers a seamless solution for customizing your WordPress admin panel to suit your preferences. With the settings conveniently located under Appearance → Admin Menu Settings, you have the power to tailor your admin experience effortlessly.
Key Features:

1. Instant Menu Concealment: Upon activation, the plugin allows you to immediately hide menus, ensuring instant application of your customization choices.
2. Child Menu Hiding: Enable the option to hide child menus, providing a cleaner and more focused navigation experience.
3. Global or User-Specific Settings: Choose between global settings for all users or empower each user to personalize their admin menu visibility.

<hr>

* <b>Global Settings:</b> Changes apply to everyone. If the admin hides a menu, it's hidden for all users.
* <b>User-Specific Settings:</b> Each user can customize their menu visibility individually.<hr>

Enhance your WordPress admin panel experience by effortlessly managing menu items and optimizing the interface according to your specific needs. Experience a clutter-free and user-friendly environment with <u>Easy Hide Admin Menu Items</u>.

== Installation ==

You can install this plugin in two ways:

1. **Installation via Uploader:**
   - Download the plugin archive and extract it.
   - Upload the `easy-hide-admin-menu-items` folder to the `wp-content/plugins/` directory on your server.
   - Activate the plugin through the "Plugins" section in the WordPress admin panel.

2. **Installation from Repository:**
   - Log in to your WordPress admin panel.
   - Go to the "Plugins" section and select "Add New".
   - Enter the <b><u>`EHAMI`</u></b> key for simplified searching in the search bar.
   - Click the "Install" button next to the "Easy Hide Admin Menu Items" plugin.
   - After the installation is complete, activate the plugin.

### Frequently Asked Questions

**Q: How to hide a menu?**
A: After installing the plugin, a control manager will appear in the admin bar; you can disable the menu at any time.

**Q: Can I make the menu hide for each user?**
A: Yes, you can. We introduced an additional settings page that we will improve over time. Please send your suggestions to [support](https://wordpress.org/support/plugin/easy-hide-admin-menu-items/).
